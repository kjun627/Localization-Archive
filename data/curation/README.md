# Curation Files

Add one YAML file per included paper.

Required fields:

- `paper_id`
- `title`
- `year`
- `venue`
- `venue_type`
- `venue_tier`
- `problem`
- `prior_gap`

Recommended fields:

- `summary`
- `metric`
- `why_this_metric`
- `dataset`
- `dataset_limitations`
- `limitations`
- `source_links`
- `provenance`
- `confidence`
